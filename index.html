Mapzap.me

Build A Maps App

A simple, lightweight tool for quickly standing up responsive web mapping applications without having to write any code! Built on the Google Maps JavaScript API, this utility lets you configure your settings via simple URL parameters.

Start Building!

Examples

  • DC WiFi Social - A GitHub based collaborative list of DC locations that serve up both Internet and Alcohol from Ben Balter
  • US States - Natural Earth admin 1 states provinces shp data from geojson.xyz
  • Football Clubs of Europe - A Fulcrum mapping project from Coleman McCormick
  • NYC Coffee Shops - via Mapbox
  • Baseball Parks - A GeoJSON Ballpark mapping project from James Fee

Fast & Easy

Define app configuration parameters manually or with the handy Builder Tool.

GeoJSON & CSV

Data source can be any web accessible GeoJSON or CSV (with latitude & longitude columns) file.

Responsive

Generate a fullscreen mobile-friendly web app with responsive navbar, modal popups, and map/table/split views.

Serverless

Completely client-side, can be hosted for free on GitHub Pages.

Customizable

Set the app title, icon, display fields, data attribution & styles for a custom experience.

Best In Class

Built on the incredibly popular Bootstrap and Google Maps frameworks.

Interactive

Interactive feature table with filtering, sorting, and column toggling via Bootstrap Table.

Open Source

You can download or fork the code repository and host your own version. If you self-host, please use your own Google Maps API key.
